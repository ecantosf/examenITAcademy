# ITAcademy-examen
A simple README for the IT Academy exam resolution.

Historias de usuario
Registro de usuario
Como

nuevo usuario

quiero

registrarme en el sistema con un nombre de usuario y una contraseña

para

poder iniciar sesión más adelante.

Criterios de aceptación:

[ ]  No se debe permitir registrar un usuario con nombre vacío.
[ ]  No se debe permitir registrar un usuario ya existente.
[ ]  La contraseña debe tener al menos 8 caracteres.

Tareas (ejemplo):

[ ]  Implementar una clase User que almacene el nombre de usuario y la contraseña.
[ ]  Implementar una clase UserService con un método register(String username, String password).
[ ]  Hacer que el método cumpla con los criterios de aceptación anteriores.
[ ]  Hacer tests que cubran tanto casos de ‘happy path’ como de ‘sad path’, tales como:
[ ]  Registro con nombre vacío.
[ ]  Registro con usuario repetido.
[ ]  Registro con contraseña demasiado corta.
[ ]  Registro correcto con credenciales válidas.
